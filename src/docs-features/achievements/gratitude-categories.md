# Категории благодарностей и достижения

## 5 категорий

| Slug         | Название            | Emoji | Описание                                                     |
| ------------ | ------------------- | ----- | ------------------------------------------------------------ |
| `help`       | Помощь и поддержка  | 🤝    | Отзывчивость, готовность помочь, поддержка в работе          |
| `quality`    | Профессионализм     | ⭐    | Качественная работа, оперативность, внимание к деталям       |
| `mentoring`  | Наставничество      | 📚    | Обучение, менторство, лекции, передача опыта, консультации   |
| `teamwork`   | Командная работа    | 🛡️    | Взаимодействие со смежниками, слаженная работа, коммуникация |
| `atmosphere` | Позитив и атмосфера | ☀️    | Настроение, мероприятия, тимбилдинг, забота о коллективе     |

Категория выбирается при отправке благодарности. 💎 одинаковые для всех категорий (20 ПК).

## Достижения (3 из 5 категорий)

| Достижение     | Категория   | Порог                    | Бонус         |
| -------------- | ----------- | ------------------------ | ------------- |
| Надёжное плечо | `help`      | 4 благодарности за месяц | ПК (обсудить) |
| Эксперт        | `quality`   | 4 благодарности за месяц | ПК (обсудить) |
| Наставник      | `mentoring` | 4 благодарности за месяц | ПК (обсудить) |

Категории `teamwork` и `atmosphere` не дают достижений — только статистика.

## Механика

- Период = календарный месяц (как Revit/WS достижения)
- Пороговое: получил 4 благодарности в категории = достижение
- Не зависит от топ-рейтинга (в отличие от Revit/WS)
- Кубок на странице /achievements
- Бонус начисляется автоматически через gamification_event_logs
